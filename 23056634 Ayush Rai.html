5 Statical Testing

Test 1 : Whether the average response time across complaint types is similar or not.

We will be performing One-Way ANOVA Test because:

Stating the hypothesis

Null hypothesis (H0) = The average response time across all the complaint types is the similar

Example:

Let 'X' be the average closing time for complaint types, and let 5 be the total number of complaints.

Then, according to the Null Hypothesis, all of the average closing times is similar

a * X1 = b * X2 = c * X3 = d * X4 = e * X5 (a,b,c,d,e are negligible constants)

Alternate hypothesis (H1) = The average response time across complaint types is not similar

Example:

Let 'X' be the average closing time for complaint types, and let 3 be the total number of complaints.

Then, according to the Alternate Hypothesis, at least one of the average closing time is not equal

Either: X1 != X2 or X2 != X3 or X1 != X3

In[402]:
# Grouping by Complaint Type

complaint_groups = dataSet.groupby('Complaint Type')['Request_Closing_Time_Int']

# Performing One-way ANOVA

f_stat, p_value = stats.f_oneway(*[group for name, group in complaint_groups])
print(f'We got ANOVA F-statistic: {f_stat}')
print(f'We got P-value: {p_value}')

print("")
# Checking the output

if p_value <= 0.05:
    print('We Accept the Alternative hypothesis, The average response time across all the complaint types is not similar')
else:
    print('We Accept the Null hypothesis, The average response time across complaint types is similar')

Test 2: Whether the type of complaint or service requested and location are related.

Here, we will be performing the Chi-Square Test of independence because:

Both variables (complaint type and location) are categorical.

We want to find the association between the two categorical variables.

Stating the hypothesis:

Null Hypothesis (H0): The complaint type and location are not related

Alternate Hypothesis (H1): The complaint type and location are related

Performing Chi-Square Test of Independence

In[403]:
# Create a contingency table (cross-tabulation) between Complaint Type and Location

table = pd.crosstab(dataSet['Complaint Type'], dataSet['Borough'])

# Perform the Chi-Square Test of Independence
chi2_stat, p_value, dof, expected = stats.chi2_contingency(table)

if p_value <= 0.05:
